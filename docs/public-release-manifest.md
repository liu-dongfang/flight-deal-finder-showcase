# Public Release Manifest

## Retained In This Showcase Repo

- The runnable Next.js prototype for discounted-flight discovery
- Deterministic local route, fare, and calendar data used by the demo
- UI logic for comparison, filtering, calendar shifting, and the detail drawer
- Public-facing README, product documentation, and architecture notes
- Repository screenshots exported from the current product interface

## Intentionally Excluded From The Public Repo

- `.env` files, tokens, and local machine configuration
- `node_modules`, `.next`, `.vercel`, and other local build or deployment leftovers
- Internal drafting notes, assessment-specific working files, and local QA scratch materials
- Any claim of real supplier integration, live pricing infrastructure, or private business logic not present in this prototype

## Mock / Synthetic / Demo-Only Content

- Flight results are deterministic demo data, not live airline inventory
- Calendar price shifts are synthetic examples designed to make the decision flow stable and reviewable
- AI-style recommendation copy is generated from local rules and seeded presentation logic
- The repository shows product behavior for a constrained set of routes and scenarios rather than full market coverage

## Public Positioning

This repo should be read as a public product prototype showcase. It demonstrates product framing, interface design, and deterministic delivery quality. It should not be read as a claim of real booking capability, production OTA infrastructure, or live airline integration.
